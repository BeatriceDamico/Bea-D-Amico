Coin Booklet

This card set is designated for the future, when coins become extinct, dominated by different versions of a "credit card." The coin cards display ways coins were used with machines for the machine to function. The coin slot is built so the user can collect coins by inserting the coin through the cards and all coins are collected in the bottom of the card box.
